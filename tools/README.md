# ReND Tools

This folder contains development tools used to maintain the ReND project.

## audit

Scripts that analyze the knowledge base.

Examples:
- alias_audit.py
- full_knowledge_audit.py

## generators

Scripts that generate new data.

Examples:
- generate_aliases.py

## maintenance

Scripts that fix or clean existing data.

Examples:
- fix_duplicate_aliases.py
- fix_duplicate_questions.py

## migrations

One-time migration scripts.

Examples:
- auto_pack_migration.py

## importers

Scripts that import external knowledge into ReND.
